# A25 浏览器扩展注入壳

这个目录是“真实接入超星 / 智慧树页面”的最小可用方案，不依赖真实 AI，只负责把嵌入式助教面板注入到页面里。

## 使用方式

1. 先启动后端和前端开发服务
2. 打开 Chrome 的扩展管理页
3. 开启开发者模式
4. 选择“加载已解压的扩展程序”
5. 选中 `/Users/xrz/Home/Code/服创赛/extension`

## 当前行为

- 命中 `*.chaoxing.com` 或 `*.zhihuishu.com` 页面时，会注入右下角悬浮按钮
- 点击按钮后，右侧弹出 iframe 助教面板
- 助教面板加载的是 `http://127.0.0.1:5174/`
- 聊天、课程和练习数据走本地 `http://127.0.0.1:8000` mock API

## 后续接 CopilotKit 的建议

当前这层扩展和 iframe 入口可以保持不变，后面只要把 `widget-react` 内部交互层替换成 CopilotKit 驱动的实现即可。
